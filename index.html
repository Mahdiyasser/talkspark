API Documentation

Getting Started

The TalkSpark API provides programmatic access to our entire collection of conversation starters. All responses are in JSON format.

Base URL: /api/

Endpoints

Random Topic

/?talk

Returns a random conversation starter from any category.

Random from Category

/?c={category-id}

Returns a random topic from the specified category.

Example: /?c=9 (Business & Money)

Random from Multiple Categories

/?c={id1}&{id2}&{id3}

Returns a random topic from the specified categories.

Example: /?c=4&9&10

Specific Point

/?c={category-id}&&p={point-id}

Returns a specific conversation starter.

Example: /?c=9&&p=5

Random from Specific Points

/?c={category-id}&&p={id1}&{id2}&{id3}

Returns a random topic from specific points within a category.

Example: /?c=9&&p=1&5&10

Multiple Categories with Specific Points

/?mc=yes&&c1={cat-id}&{p-id}&{p-id}&&c2={cat-id}&{p-id}

Returns a random topic from specific points across multiple categories.

Example: /?mc=yes&&c1=9&1&5&&c2=4&3&7

Search All Categories

/?search={keyword}

Searches all topics for the keyword in the name.

Example: /?search=AI

Search Specific Category

/?search={keyword}&&c={category-id}

Searches topics within a specific category.

Example: /?search=work&&c=9

Search Multiple Categories

/?search={keyword}&&mc=yes&&c1={id}&&c2={id}

Searches topics within multiple categories.

Example: /?search=future&&mc=yes&&c1=5&&c2=10

Multi-Keyword Search

/?search={keyword1}|{keyword2}|{keyword3}

Searches for topics matching any of the keywords (OR search).

Example: /?search=AI|robot|future

Multi-Keyword Search in Categories

/?search={kw1}|{kw2}&&mc=yes&&c1={id}&&c2={id}

Searches for multiple keywords within specific categories.

Example: /?search=money|work&&mc=yes&&c1=9&&c2=13

Response Format

All endpoints return JSON with the following structure:

{
  "id": 1,
  "name": "Topic Name",
  "the-point": "The conversation starter or question",
  "context": "Additional context",
  "category": "Category Name",
  "category_id": 1
}
